# OnePlus-bbs-Bot

Automatic Signer/Drawer Bot. ver 1.3.1

Use for OnePlus bbs (CN).

Replace `cookie` with your bbs cookies.

## Usage

- Webpage:

PHP + nginx: `do.php?mode=[sign|draw|all]`

- Command:

linux: `php do.php [sign|draw|all]`

windows: `php.exe do.php [sign|draw|all]`
